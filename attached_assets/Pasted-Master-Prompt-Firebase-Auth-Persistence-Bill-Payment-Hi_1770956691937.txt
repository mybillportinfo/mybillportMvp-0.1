## FEATURE 2: BILL PAYMENT HISTORY & “MARK AS PAID”

**Goal:** Users can manually mark bills as paid, and the app stores a history of payments for each bill.

**Firestore Schema Changes:**

**1. Bills collection document – add these fields:**

```javascript
{
  // ... existing fields (billerName, amount, dueDate, etc.)
  status: 'upcoming' | 'paid' | 'overdue',  // default 'upcoming'
  paidAt: Timestamp | null,                // default null
  lastPaymentAmount: number | null,
  lastPaymentDate: Timestamp | null
}
```

2. New subcollection: bills/{billId}/payments

Each payment document:

```javascript
{
  paidAt: Timestamp,          // server timestamp
  amount: number,            // snapshot of bill amount at time of payment
  method: 'online' | 'mail' | 'in‑person' | null,
  confirmationCode: string,  // optional, user can enter
  recordedVia: 'manual',     // or later 'email_forward'
  notes: string
}
```
