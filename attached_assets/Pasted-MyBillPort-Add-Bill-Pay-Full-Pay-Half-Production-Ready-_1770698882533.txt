3️⃣ PAY FULL / PAY HALF — CORE FEATURE

On Home / Bills list, every unpaid bill must show:

Buttons
	•	Pay Full
	•	Pay Half

Logic (IMPORTANT)

This is NOT real payment. It is state tracking.

Pay Full
	•	paidAmount = amount
	•	remainingAmount = 0
	•	status = paid
	•	Disable buttons
	•	Show “Paid”

Pay Half
	•	paidAmount = amount / 2
	•	remainingAmount = amount / 2
	•	status = partially_paid
	•	Button changes to “Pay Remaining”

Pay Remaining
	•	Clears remaining amount
	•	Sets status to paid

All updates persist in Firestore.
